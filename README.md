# Compass Ribbons v0.0.0

A scss mixin (compass) for simply generate css3 ribbons.
By Naivencreatik

## Usage
import the "ribbons" file

```scss
@include ribbon(...);
```
// Define short $params


##Example 
```scss
@import "ribbons";

.box{
	@include ribbon(...);
}
```
